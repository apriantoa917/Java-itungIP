Panduan menggunakan aplikasi itung IP

1. pastikan device yang anda gunakan terinstal aplikasi XAMPP dan Java versi diatas 2.XX
2. buat database baru di phpMyAdmin dengan nama ' nilai '
3. Nyalakan MySQL pada aplikasi XAMPP
4. Masuk kedalam phpMyAdmin dan pilih database nilai dan masuk ke tab import dan lakukan import data berupa file SQL dalam folder ini
5. Aplikasi siap digunakan
